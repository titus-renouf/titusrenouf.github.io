Google Cloud Platform

This is my Cloud Storage page. Here you can see folders named "buckets". Each bucket corresponds to a storage location where you can upload some data. The goal of this project is to upload a dataset in the "bronze-titus" bucket, then to clean it into the "silver-titus" bucket using Cloud Function to be able to upload it into Google BigQuery.


This is the bronze-titus bucket page, where you can find the bucket configuration. As I am located in France I chose to set the zone to europe-west to respect the RGPD obligations. By using a Cloud Function I uploaded two versions of a dataset about data scrapping on anime series, one in CSV and an other one in JSON.


This is an other bucket page where you can see a file called "First_cleaned_version.json" corresponding to the JSON of my bucket "bronze-titus" that I have cleaned using Cloud Function again.


This is the list of our Cloud Functions corresponding to the specific project. I chose to trigger them using a HTTP request to facilitate the test phase, but it could be interesting in our case to set event triggers.


This is the deployment history of our cleaning function. We can see that a column vote has been well created, such as the rest of the cleaning (rank format, rating format, number of episodes, ...).


This is the BigQuery page where we succeed to import our cleaned version of the dataset. So now we are able to create requests in order to answer some asked questions.

I will probably start again this project to improve my cloud functions and then pursue it till the creation of a Business Intelligence tool, but it is quite enough for a first familiarization with this new technology.
