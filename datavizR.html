Data Visualization using R

Comparing Decathlon scores :

For this project we will explore visualization possibilities using R applied on a Decathlon results data frame.





We are box plotting the score according to the age category. We can say that the U19 category got better scores than U17, but the highest score is detained by a U17.
It also seems that there have been an abandonment for the U17 category.






We are plotting the link between discus throwing and shot put score. To analyze it we also precised the club because the athletes are not trained the same way.
We can see that there is a positive correlation between the two events scores.









Contrary to the previous plot, here we can see that there is no correlation between long jumping and high jumping scores. We just computed another type of plot.

Analyzing data correlations with PCA :

This project goal is to familiarize with PCA analyzing technic on R. We chose to applied it on a food composition data frame and especially chose to work on the cheeseburger composition.





The PCA enables us to see the burger components impact on the ecological system. We can see that "Floor", "Butter", "Other steps" and "Cheddar cheese" are impacting badly the ecological resources. But the chedar cheese is above all, indeed we want to study if we could exchange it with a less impactful cheese.






This second PCA plot, comparing cheeses impact, shows us that cheddar could be substitute to Gorgonzola as it is opposed to the majority of ecological resources depletion.
In facts, it is obviously not the best idea to exchange cheese as all of them have a negative impact on ecological resources, the ideal idea would be to avoid it. But that was a way to touch on a problem using PCA.
