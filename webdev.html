React Native is a framework created by Facebook in 2015. It is positioned as a cross-platform while using native mobile components. This way the framework adopts its own language: a React Native view will turn into UIView for IOS and android.view.View for Android. Thus, React Native turns out to be the perfect consensus between native and cross-platform. He becomes the mobile development tool par excellence, for example Instagram, Uber Eats or AirBnb were developed in React Native.
